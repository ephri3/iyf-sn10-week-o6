# Week 6: Asynchronous JavaScript - Weather Dashboard

## Overview
This project demonstrates mastery of **asynchronous programming in JavaScript**. Users can view real-time weather data fetched from the OpenWeatherMap API, while learning how to work with callbacks, Promises, async/await, and handle API responses gracefully.

## Features
- Fetch live weather data using **OpenWeatherMap API**
- Handle asynchronous operations with:
  - Callbacks
  - Promises and Promise chaining
  - Async/Await
- Error handling for failed API requests
- Loading states to enhance user experience
- Responsive and dynamic updates to the DOM

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- OpenWeatherMap API
- DOM Manipulation
